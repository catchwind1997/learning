# 什么是Git？

Git是一个分布式文件版本管理系统。“版本管理系统”是一种能追踪文件更改，保存文件历史版本的系统。
借助Git，能够方便地回顾追踪的文件的每一次修改，还可以方便地还原文件到某一个特定的版本。

要使用Git，首先要理解仓库的概念。

仓库是保存所有文件的位置。只有被放到仓库中的文件才能被Git跟踪。

用户可以把自己在一台电脑上的的文件推送到远端，以便自己或者其它人在另外的电脑上克隆远端上的仓库。
